Snippet

<script
  src="https://github.com/pavelsima/Answerlay-embed/releases/latest/download/embed.js"
  data-assistant-id="YOUR_ASSISTANT_ID"
  defer
></script>

Local fixture demo

Pages that don't have a real published config can still render the widget by passing data-config-url to point at a local file. That's what this example does — no <answerlay-chat> tag, just the script. Dev mode uses type="module" because Vite serves the source as ESM; the production bundle is IIFE and does not need it.
